## Launch Checklist

- [ ] Verify conversion tracking tags (Sign-up, App install)
- [ ] Confirm landing page loads fast (< 3s) on mobile and desktop
- [ ] Add UTM parameters to all final URLs for analytics
- [ ] Set up Google Analytics + Google Ads linking
- [ ] Review keyword match types; add negative keywords per above
- [ ] Enable ad rotation: "Rotate indefinitely" (don't let Google optimize early)
- [ ] Set campaign start date and end date (if applicable)
- [ ] Configure budget alerts at 80% spend
- [ ] Run ad previews and check for disapprovals
- [ ] Schedule weekly review of search term reports to add new negatives
- [ ] After 2 weeks: pause low-performers, shift budget to winners
- [ ] Consider separate "Brand" campaign if "Heard Again" brand searches appear
