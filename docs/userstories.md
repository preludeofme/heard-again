# Heard Again - User Stories

## 🧱 1. Authentication & Accounts

### Core
- [ ] As a user, I want to create an account so I can save and access my family data
- [ ] As a user, I want to log in to my account so I can access my family tree
- [ ] As a user, I want to log out of my account to protect my data
- [ ] As a user, I want to reset my password if I forget it

### Optional
- [ ] As a user, I want to log in with Google or Apple for convenience
- [ ] As a user, I want to manage my profile (name, email, avatar)

---

## 👨‍👩‍👧‍👦 2. Family Tree & People Management

### Core
- [ ] As a user, I want to create a family tree so I can organize my family
- [ ] As a user, I want to add family members to my family tree
- [ ] As a user, I want to edit a family member’s details (name, birthdate, relationship)
- [ ] As a user, I want to delete a family member
- [ ] As a user, I want to define relationships between members (parent, sibling, spouse)

### Visualization
- [ ] As a user, I want to view my family tree in a visual layout
- [ ] As a user, I want to navigate large family trees easily (zoom/pan)

### Extended Network
- [ ] As a user, I want to add non-family people (friends, mentors) to my network
- [ ] As a user, I want to categorize people (family, friend, other)

---

## 📖 3. Stories & Memory System

### Core
- [ ] As a user, I want to create a story for a specific person
- [ ] As a user, I want to edit a story
- [ ] As a user, I want to delete a story
- [ ] As a user, I want to tag stories (childhood, holidays, advice, etc.)

### Organization
- [ ] As a user, I want to view all stories for a person
- [ ] As a user, I want to browse stories by category
- [ ] As a user, I want to search stories by keyword

### Enhancements
- [ ] As a user, I want to mark stories as favorites
- [ ] As a user, I want to create collections of stories

---

## 🎤 4. Audio & Voice Capture

### Core
- [ ] As a user, I want to upload audio recordings to a person
- [ ] As a user, I want to attach audio recordings to a story
- [ ] As a user, I want to record audio directly in the app
- [ ] As a user, I want to preview audio before saving

### Processing
- [ ] As a user, I want uploaded audio to be cleaned or processed automatically
- [ ] As a user, I want the system to extract usable voice samples from recordings

### Management
- [ ] As a user, I want to see all recordings for a person
- [ ] As a user, I want to delete recordings

---

## 🧠 5. Voice Cloning & Playback

### Voice Creation
- [ ] As a user, I want to create a voice profile from recordings
- [ ] As a user, I want to regenerate or improve a voice profile
- [ ] As a user, I want to see the status of voice creation (processing, ready)

### Playback
- [ ] As a user, I want to listen to a story in a person’s voice
- [ ] As a user, I want to replay generated audio
- [ ] As a user, I want to choose which voice to use for playback

### Controls
- [ ] As a user, I want to adjust playback speed
- [ ] As a user, I want to adjust tone or emotion (e.g., calm, excited)
- [ ] As a user, I want to regenerate audio if I don’t like it

### Advanced
- [ ] As a user, I want to generate new narrated content from text
- [ ] As a user, I want to hear stories in different emotional styles

---

## 🔐 6. Permissions, Sharing & Collaboration

### Core
- [ ] As a user, I want to share my family tree with others
- [ ] As a user, I want to invite users to view/add/edit my family tree
- [ ] As a user, I want to accept or decline an invite

### Permissions
- [ ] As a user, I want to control who can view my data
- [ ] As a user, I want to control who can edit my data
- [ ] As a user, I want to assign roles (viewer, editor, admin)

### Safety
- [ ] As a user, I want my data to be private by default
- [ ] As a user, I want to control which stories are shared

---

## 🌐 7. Self-Hosting & Connected Mode

### Local Mode
- [ ] As a user, I want to run Heard Again locally on my machine
- [ ] As a user, I want my data to remain on my local machine

### Connected Mode
- [ ] As a user, I want to connect my local instance to HeardAgain.com
- [ ] As a user, I want to access my instance remotely through a secure domain
- [ ] As a user, I want to see the status of my connection (online/offline)

### Hybrid Compute
- [ ] As a user, I want to use cloud GPU for generating audio while keeping my data local
- [ ] As a user, I want to control whether generation happens locally or in the cloud

---

## 💳 8. Billing & Usage

### Core
- [ ] As a user, I want to subscribe to a plan
- [ ] As a user, I want to view my current plan and usage
- [ ] As a user, I want to upgrade or downgrade my plan

### Usage
- [ ] As a user, I want to see how many minutes of audio I’ve generated this month
- [ ] As a user, I want to be notified when I’m nearing my limit
- [ ] As a user, I want to purchase additional usage if needed

---

## 📦 9. Import / Export

### Export
- [ ] As a user, I want to export my family tree as a PDF
- [ ] As a user, I want to export my data as a backup (JSON/ZIP)
- [ ] As a user, I want to export stories with audio

### Import
- [ ] As a user, I want to import a family tree from another source
- [ ] As a user, I want to import GEDCOM or similar formats
- [ ] As a user, I want to import audio files in bulk

---

## 🔍 10. Search & Discovery

- [ ] As a user, I want to search for family members
- [ ] As a user, I want to search for stories
- [ ] As a user, I want to filter by person, tag, or date
- [ ] As a user, I want to quickly access recent or favorite content

---

## 🛡️ 11. Trust, Consent & Safety

- [ ] As a user, I want to confirm I have permission to use someone’s voice
- [ ] As a user, I want generated audio to be clearly labeled as AI-generated
- [ ] As a user, I want to control whether a voice can be used for generation
- [ ] As a user, I want to delete a voice profile permanently
